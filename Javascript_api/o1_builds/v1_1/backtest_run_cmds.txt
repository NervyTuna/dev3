Below is the general way to run your backtest.py script on Windows, specifying which years to process:

Open either Command Prompt or PowerShell.

Navigate (cd) to the folder containing your script and CSV. For example:

bash
Copy
cd "C:\Users\stuar\OneDrive\Documents\Working\Trading\dev2\Javascript_api\o1_builds\v1_1"
Run the script with Python, pointing to your CSV and specifying a year or range via --year_range.

Examples
Single year (e.g., 2024):

bash
Copy
python backtest.py --csv "dax-1m.csv" --year_range "2024"
Range of years (e.g., 2000 to 2003):

bash
Copy
python backtest.py --csv "dax-1m.csv" --year_range "2000-2003"
Another single year (e.g., 2010):

bash
Copy
python backtest.py --csv "dax-1m.csv" --year_range "2010"
Another single year (e.g., 2020):

bash
Copy
python backtest.py --csv "dax-1m.csv" --year_range "2020"
